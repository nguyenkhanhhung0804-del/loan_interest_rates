# Loan Interest Rates Analysis

This project analyzes loan interest rate data using Python and Jupyter Notebook. The workflow includes data collection, preprocessing, exploratory data analysis (EDA), and visualization to identify trends and relationships within the dataset.

## Features

- Data loading and preprocessing
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Data visualization
- Reproducible workflow using Jupyter Notebook

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure

```
loan-interest-rates/
│── data/
│── notebooks/
│── figures/
│── loan_interest_rates.ipynb
│── requirements.txt
│── README.md
```

## Getting Started

1. Clone the repository.
2. Install the required packages.
3. Open `loan_interest_rates.ipynb` in Jupyter Notebook.
4. Run the notebook cells sequentially.

# Loan Interest Rate Analytics (Python & Statistical Data Analysis)

This repository contains a comprehensive data analysis project focused on loan interest rates, borrower characteristics, and lending risk assessment. It utilizes a consumer loan dataset (`LoanData.csv`) together with **Python** and statistical analysis techniques to extract meaningful insights through exploratory data analysis (EDA), hypothesis testing, and data visualization.

---

## 📊 Dataset Overview

The project analyzes a consumer loan dataset containing borrower financial information and loan characteristics.

The dataset consists of borrower and loan attributes including:

1. **Loan Information**

   * Includes loan amount, applied amount, interest rate, loan purpose, and loan term.
2. **Borrower Information**

   * Contains annual income, employment length, home ownership status, debt obligations, and demographic characteristics.
3. **Credit Risk Indicators**

   * Includes debt-to-income ratios, loan ratings, and other financial variables used to evaluate lending risk.

Together, these variables provide a comprehensive view of borrower profiles, loan pricing, and factors influencing interest rates.

---

## 🐍 Python Analytics & Statistical Skills

In addition to exploratory analysis, this project demonstrates a complete **Python-based analytics workflow** for transforming raw lending data into actionable financial insights. This component highlights several core data analytics capabilities:

* **Data Wrangling with pandas:** Cleaned and prepared loan data by setting indexes, handling missing values, creating derived variables, and restructuring datasets for analysis.
* **Feature Engineering:** Constructed new analytical variables, including debt-to-income ratios and borrower risk classifications, to support financial risk assessment.
* **Exploratory Data Analysis (EDA):** Leveraged **pandas** and **NumPy** to summarize loan distributions, compare borrower groups, and investigate factors associated with loan interest rates.
* **Statistical Analysis:** Applied confidence intervals and descriptive statistical techniques using **SciPy** and **Statsmodels** to quantify uncertainty and evaluate borrower characteristics.
* **Data Visualization:** Developed informative visualizations using **Matplotlib** and **Seaborn**, including boxplots, scatter plots, histograms, and correlation analyses to communicate lending patterns and borrower behavior.

---

## 🛠️ Analytical Tasks & Business Questions

The notebook includes Python analyses designed to answer key lending and financial questions, structured into the following analytical themes:

### 1. Loan Portfolio Analysis

* **Interest Rate Distribution:** Summarizes average loan interest rates and measures overall portfolio variability.
* **Loan Amount Analysis:** Compares requested loan amounts with approved funding to identify lending patterns.
* **Borrower Segmentation:** Examines how borrower characteristics influence loan pricing.

### 2. Credit Risk Assessment

* **Debt-to-Income Analysis:** Creates debt-to-income ratios to identify potentially higher-risk borrowers.
* **Risk Classification:** Segments borrowers into risk groups and compares average interest rates across categories.
* **Financial Profile Comparison:** Evaluates relationships between borrower income, loan amounts, and lending risk.

### 3. Statistical Exploration

* **Confidence Interval Estimation:** Estimates population parameters for loan interest rates using statistical inference.
* **Correlation Analysis:** Investigates relationships between numerical borrower attributes and interest rates.
* **Distribution Analysis:** Explores the distribution of financial variables to identify trends and outliers.
